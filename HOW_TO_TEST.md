## 🎮 Тестирование в Telegram

1. Запусти бота:
```bash
python main.py
```

2. Отправь команды:
   - `/start` → должно быть приветствие
   - `/shop` → должны быть товары
   - `/random` → должен быть случайный товар

3. Проверь кнопки:
   - Нажми "Далее" → следующие товары
   - Нажми "Назад" → предыдущие товары

## 🐛 Если что-то не работает

- Проверь интернет
- Перезагрузи бота (Ctrl+C → python main.py)
- Смотри ошибку в консоли
